# Snowflake & AWS S3 Data Pipeline

## Overview
End-to-end automated data pipeline built using Snowflake and AWS S3 for continuous data ingestion and transformation.

## Dataset
Superstore Sales Data (Kaggle)

## Tools & Technologies
- Snowflake
- AWS S3
- SQL
- Snowpipe
- Streams & Tasks

## Pipeline Architecture
1. Raw data stored in AWS S3 bucket
2. Snowpipe for automated continuous ingestion
3. Staging and transformation layers for data cleaning
4. Streams & Tasks for incremental processing
5. Clean analytics-ready table for reporting

## Features
- Automated ingestion pipeline
- Incremental data processing
- Data cleaning and transformation
- Structured reporting-ready tables

## Learning Outcomes
- Understanding ETL workflows
- Snowflake data warehousing concepts
- Cloud-based data integration
- Incremental data handling using Streams & Tasks
